Contexte de l'entreprise

Télétravail intégral — Pau, 64

ARGITIK est une entreprise basée à Pau (Pyrénées-Atlantiques) proposant des services numériques : développement web, conseil informatique et accompagnement à la transition digitale pour des PME et TPE locales.

La mission confiée lors de ce stage portait sur la réalisation complète d'un site web vitrine avec formulaire de contact fonctionnel, incluant l'envoi automatique des données saisies par e-mail vers une adresse Outlook, via un script PHP et un serveur SMTP.

L'intégralité du stage s'est déroulée en télétravail, ce qui a impliqué une organisation rigoureuse, des points réguliers avec le maître de stage et une communication écrite structurée pour valider chaque livrable.

Mission & Problématique

"ARGITIK a besoin d'un site web vitrine permettant à ses prospects de prendre contact en ligne. Comment concevoir et développer un site en HTML / CSS, avec un formulaire de contact traité en PHP qui transmet automatiquement les données saisies vers une boîte mail Outlook via SMTP, tout en garantissant la validation des champs et la sécurité des envois ?"

Architecture de la solution

Outils & Technologies

HTML5
Structure sémantique du site vitrine : pages, sections, formulaire de contact avec attributs de validation natifs.
CSS3
Mise en page responsive, animations, variables CSS. Design adapté à l'identité visuelle d'ARGITIK et compatible mobile.
PHP
Traitement côté serveur : réception des données POST, validation, sécurisation et envoi e-mail via mail() / SMTP.
SMTP / Outlook
Configuration de l'envoi automatique vers une boîte Microsoft Outlook via protocole SMTP avec authentification TLS.
Visual Studio Code
Éditeur principal : extensions HTML/CSS/PHP, Live Server pour le rendu en temps réel, terminal intégré.
Serveur web local
Environnement de test PHP en local (WAMP/XAMPP) pour valider le traitement du formulaire avant déploiement.

Exemple — Script PHP d'envoi du formulaire

PHP · Traitement formulaire de contact · envoi SMTP
<?php // Traitement du formulaire de contact — envoi vers Outlook if ($_SERVER['REQUEST_METHOD'] === 'POST') { // Sécurisation des entrées $nom = htmlspecialchars(strip_tags(trim($_POST['nom']))); $email = filter_var(trim($_POST['email']), FILTER_SANITIZE_EMAIL); $sujet = htmlspecialchars(strip_tags(trim($_POST['sujet']))); $message= htmlspecialchars(strip_tags(trim($_POST['message']))); // Validation if (!filter_var($email, FILTER_VALIDATE_EMAIL)) { $erreur = "Adresse e-mail invalide."; } elseif (empty($nom) || empty($message)) { $erreur = "Tous les champs sont requis."; } else { // Construction du message $destinataire = "contact@argitik.fr"; $en_tete = "From: $nom <$email>\r\n"; $en_tete .= "Reply-To: $email\r\n"; $en_tete .= "Content-Type: text/plain; charset=UTF-8\r\n"; $corps = "Nouveau message de : $nom\n" . "E-mail : $email\n\n" . "$message"; // Envoi via fonction mail() configurée SMTP (php.ini) if (mail($destinataire, $sujet, $corps, $en_tete)) { $succes = "Message envoyé avec succès !"; } } } ?>

Déroulement du stage

S1
Semaine 1
Prise de contact & analyse du besoin
Réunion de lancement (visio) avec le maître de stage pour définir le périmètre du projet. Analyse du besoin client : pages souhaitées, charte graphique, fonctionnalités du formulaire, adresse de destination Outlook. Configuration de l'environnement de développement local (WAMP, VS Code).
S2
Semaine 2
Maquettage & structure HTML
Conception des maquettes des pages (accueil, services, contact) validées avec le maître de stage. Développement de la structure HTML5 sémantique : header, navigation, sections, footer. Mise en place de l'architecture des fichiers du projet.
S3
Semaine 3
Intégration CSS & responsive design
Développement du CSS : mise en page avec Flexbox/Grid, typographie, couleurs, variables CSS, animations légères. Adaptation responsive pour mobile, tablette et desktop. Validation visuelle sur différents navigateurs (Chrome, Firefox, Edge).
S4
Semaine 4
Développement du formulaire PHP
Développement du script PHP de traitement : réception des données POST, validation avec filter_var(), sécurisation contre les injections e-mail (htmlspecialchars()), et envoi via la fonction mail() configurée avec SMTP.
S5
Semaine 5
Configuration SMTP & tests d'envoi Outlook
Configuration du serveur SMTP dans php.ini pour acheminer les e-mails vers la boîte Outlook d'ARGITIK. Tests d'envoi end-to-end, vérification de la réception, correction des problèmes de chiffrement TLS et des en-têtes e-mail.
S6
Semaine 6
Tests finaux, livraison & documentation
Tests complets (formulaire, responsive, navigateurs, scénarios d'erreur), corrections finales. Livraison des fichiers sources au client, rédaction de la documentation d'installation et d'utilisation. Présentation du rendu final lors d'un point visio de clôture.
